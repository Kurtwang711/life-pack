### 底部导航栏
**使用统一的CustomBottomNavigation组件**:

#### 布局参数
- **位置**: bottom: 30px 悬浮
- **组件**: `CustomBottomNavigation`
- **功能**: 
  - 首页按钮: 导航回首页
  - 个人中心按钮: 暂时只打印日志

#### 导航逻辑
```dart
CustomBottomNavigation(
  currentIndex: _currentNavIndex,
  onTap: (index) {
    if (index == 0) {
      _navigateToHome(); // 返回首页
    } else if (index == 1) {
      print('个人中心功能开发中...'); // 占位逻辑
    }
  },
)
```
